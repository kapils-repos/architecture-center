# Open Studio for Data Quality

## Introduction

Open Studio for Data Quality provides ability to assess your data quality by leveraging the data profiling capabilities. It is a free data quality tool that profiles your data and provides a graphical drill down of the details, so you can match up to data sources and apply business rules to make sure it fits. ## Architecture

From Talend Open Studio for Data Quality, users can access and examine the data available in different data sources and collect statistics and information about this data.
